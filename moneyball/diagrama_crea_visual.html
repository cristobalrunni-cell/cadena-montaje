Flujo de crea_visual.js

0 Query Notion

Busca en el Calendario Social Media todas las paginas con Estado = "Guion" para la semana objetivo. Filtra por perfil y rango de fechas.

Notion API Estado: Guion
0

1 Leer Guion + Estilo

Lee el guion narrativo completo de la pagina Notion. Extrae los beats (escenas). Determina el pilar y carga el Manual de Estilo del creador referente.

Notion Blocks API ESTILO_*.md
1

2 Biblia Visual

Envia todos los beats a Gemini para generar un JSON con el contexto global: protagonista (edad, look), localizaciones, paleta de colores, iluminacion, estilo de edicion y tono.

gemini-2.0-flash JSON output
2

3 Procesar Escenas

Para cada beat del guion: busqueda semantica en la BBDD de 22K+ escenas, adaptacion visual con contexto global, y generacion de storyboard AI. Ver detalle abajo.

SQLite 22K+ escenas Embeddings gemini-2.5-flash-image
3

4 Generar PDF

Genera un HTML completo con la Biblia Visual + todas las escenas (guion, frames de referencia, que grabar, edicion, elementos visuales, storyboard AI). Convierte a PDF con Puppeteer.

Puppeteer HTML → PDF A4
4

5 Subir a Drive

Sube el PDF a Google Drive en la carpeta CEO Media/Storyboards/YYYY-MM. Crea carpetas si no existen. Genera link publico.

Google Drive API Service Account
5

6 Actualizar Notion

Escribe el link del PDF en la pagina Notion (debajo del guion). Cambia el estado de "Guion" → "Visual".

Notion API PATCH Estado: Visual
6

Detalle Fase 3 — Por cada beat del guion

1
Busqueda Semantica en BBDD

Genera embedding del texto del beat con gemini-embedding-001. Busca la escena mas similar del creador referente (cosine similarity). Filtra por posicion (hook/medio/final). Deduplicacion: max 2 usos, sin repetir consecutivas.

↓
F
Obtener Frames de Referencia

Busca en escenas_frames/video_XXX/ los fotogramas reales de la escena encontrada (hasta 4 frames). Se incluyen en el PDF como referencia visual.

↓
2
Adaptacion de Escena

Gemini 2.0 Flash adapta la escena de referencia al guion actual usando la Biblia Visual + Manual de Estilo. Output JSON: localizacion, accion, encuadre, cortes, elementos visuales (tipo, contenido, animacion).

↓
3
Generar Storyboard AI

Gemini 2.5 Flash Image genera una imagen de storyboard: formato 16:9 con 4 paneles verticales 9:16 (sketch a lapiz B&N). Subtitulos progresivos con keywords destacadas en color.

Estilos de Edicion

Victor Heras

Keywords visuales (1-5 palabras). NO subtitulos literales. Grafismos, iconos, contadores. Sans-serif blanco. Animacion sincronizada.

~1000 escenas

Veneno

Subtitulos COMPLETOS del dialogo. Blanco, inferior centrado. Pantalla dividida 50/50. Ritmo lento y conversacional. Glitch en tension.

2,153 escenas

Nude Project

Texto GRANDE bold (1-4 palabras). Cortes ultra-rapidos (<2s). Loading bars, typing effect. Estetica premium. Muchas escenas SIN texto.

3,089 escenas

Uso
# Semana por defecto (proximo lunes)
node crea_visual.js

# Semana especifica
node crea_visual.js --semana=2026-02-24

# Solo un pilar
node crea_visual.js --pilar=veneno

# Modo preview sin cambios
node crea_visual.js --dry-run

# Con contexto adicional
node crea_visual.js --context="enfoque cinematografico"
